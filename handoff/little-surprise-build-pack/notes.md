# Little Surprise Build Pack Notes

Use this folder as the handoff package for PA / website builder.

## How to use

1. Read `little-surprise-pa-build-prompt.md` first.
2. Inspect every image in `references/` before building.
3. Treat the images in `references/` as the primary visual design references.
4. Use Little Surprise public business info from the prompt for content.
5. Do not copy unrelated business names/text from the design references unless Sloppy explicitly says so.
6. Match the reference design closely, then verify in browser.

## Included design references

- `references/design-01-product-detail.jpg` — product detail page reference. Match the product image gallery, price/details column, quantity controls, Add to Cart / Buy Now buttons, info strip, and tabbed product-description layout.
- `references/design-02-product-catalogue.jpg` — product catalogue page reference. Match the top announcement/nav bars, banner hero, filter sidebar, product grid cards, badges, sort dropdown, load-more button, and bottom trust/contact strip.
- `references/design-03-homepage.jpg` — homepage reference. Treat this as the primary homepage visual reference. Match the header, hero composition, service/trust icons, popular-products section, CTA button, and bottom trust/contact strip.
